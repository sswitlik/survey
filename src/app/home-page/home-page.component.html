Welcome to Satisfaction Survey

• The first step is to create a template. Here we can choose which answer options will be available in the event.

• After creating the template it is possible to create a tag. It groups the events so that they can be easily searched.

• In the next step we create an event. We give it a title, description, date from, date to and have to select the template and tag that we have created before.

• If we want our event to be cyclic, we have to mark the checkbox "Cyclic event". It has also some additional options:

‣ "date first stop" - we set the end date of the single events that are included in a cyclic event,

‣ "interval" - we choose how often the events should take place,

‣ "Select blocked week days" - the days on which the event is not active,

‣ a calendar with the table "Date" - we select specific days on which the event is not active.

• On the page "event-show" we can see created events and the votes cast after clicking on the event.
